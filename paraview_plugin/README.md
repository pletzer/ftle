# Loading the Paraview plugin

This directory contains plugins that can be loaded int Paraview. 

## How to load a plugin

`Tools` -> `Manage plugins...`, then press `Load Newn`, navigate to the directory where your plugin resides. Select the plugin and press `OK`. Close the `PLgin Manager` window. 

## How to invoke the plugin

Assuming the plugin is a `Source`, go to `Sources` and select your plugin. You might have to press `Apply` to see the plugin.
